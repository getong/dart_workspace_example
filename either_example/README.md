A `dart_either` command-line example inspired by the route/catalog structure in
`flutter_row_example`.

## Run

```bash
dart run
```

This prints a home catalog with dynamic routes:

- `/examples/parse-user-id`
- `/examples/split-budget`
- `/examples/normalize-email`
- `/examples/recover-timeout`

Run one demo:

```bash
dart run bin/either_example.dart /examples/split-budget
```

You can also pass just the slug:

```bash
dart run bin/either_example.dart split-budget
```
